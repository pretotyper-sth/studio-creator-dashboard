Q1현재 활성 창작 크리에이터는 몇 명인가?

최근 90일 활성 창작 크리에이터 추세

각 지점 = 해당 월 말일 기준 최근 90일 슬라이딩 값 (마지막 지점은 오늘 기준 · 점선 = 진행 중)

주간 신규 크리에이터 (Weekly New)

최근 7일 슬라이딩 · 최초 편집 완료자

월간 Studio 활성 사용자 (MAU)

퍼블리시 여부와 무관하게 Studio를 실제로 사용한 월간 유니크

Q2누적 크리에이터는 몇 명인가?

누적 크리에이터 단계별 현황

각 지점 = 해당 월 말일 기준 서비스 시작~현재 누적 · 가입 → 첫 편집 → 퍼블리시 3개 단계 비교

Q3크리에이터는 온보딩 퍼널의 어느 단계에서 이탈하는가?

회원가입 → 스튜디오 로그인 → 에디터 진입 → 첫 편집 → 퍼블리싱 → 지속 작업

회원가입 기준 100% · 신규 크리에이터 온보딩 퍼널

첫 편집 액션 유형 비중

첫 편집을 만든 행동이 무엇인지 · foundation / technical / script / transform

가입 코호트 활성화율 (7일 내 3시간+)

Activation Rate · 퍼널 단계가 아닌 별도 활성화 지표 · 가입 후 7일 내 누적 TS ≥ 3h

Q4어느 국가에서 창작 활동이 가장 활발한가?

국가별 활성 창작 크리에이터 (최근 90일)

최근 90일 활성 창작 크리에이터의 접속 국가 기준 · 전체 세그먼트 기준(세그먼트 필터 미적용)

Q5창작 활동이 가장 활발한 크리에이터는 누구인가?

크리에이터 리더보드 (최근 90일)

활동 시간 기준 상위 크리에이터 · 전체 세그먼트 기준(세그먼트 필터 미적용)

Q6크리에이터의 퍼블리시 전환율은 얼마나 되는가?

퍼블리싱 현황

퍼블리시한 유니크 크리에이터(1명이 10개 내도 1명) + 실제 퍼블리시 월드 수 · Published(배포 상태) / Update Required 구분 · Public≠Published

월별 신규 퍼블리시 월드 (상태별)

각 지점 = 해당 월 기준(마지막은 이번 달 부분 집계) · Published(배포 완료) vs Update Required(운영 일시중지·접속불가) vs 기타 · ※ Public은 Access Setting(별도 축)

퍼블리시 월드 상태 구성 (누적)

서비스 시작~현재 누적 · World Publishing/Version Status 기준. Published=배포 완료(접속 가능). Public(Access Setting)과 다름 · 상태값 문서

퍼블리시 월드 품질 티어 분포 (누적)

숫자가 낮을수록 상위 품질 (Tier0 최상)
Tier3: 방문 700+·D1 12%+·4분+ · Tier2: 2,000+·15%+·6분+ · Tier1: 5,000+·20%+·8분+ · Tier0: 10,000+·24%+·10분+

왼쪽(낮음) → 오른쪽(높음). 초기 단계 → Tier 3 → 2 → 1 → 0

TTFP 중앙값 추이 (가입 → 첫 퍼블리시)

Time To First Publish · 코호트별 중앙값(일). 짧을수록 온보딩 효율

Q7활동 크리에이터는 늘고 있는가? (유입 vs 이탈)

인 / 아웃 — 90일 활성 크리에이터 변화

매달 말일에 "최근 90일 안에 한 번이라도 편집한 크리에이터 명단"을 만들어 직전 달 명단과 비교합니다 (마지막 지점은 오늘 기준).
진입=지난 명단에 없다가 새로 들어온 사람 · 이탈=지난 명단에 있었지만 90일간 활동이 없어 빠진 사람 · 순증 = 진입 − 이탈

순증 (진입 > 이탈) 순감 (진입 < 이탈)

Q8퍼블리시 이후에도 크리에이터는 창작 활동을 지속하는가?

정착 리텐션 — 퍼블리시 후 다시 붙어서 작업

"어느 달에 처음 월드를 낸 크리에이터들이, 그 뒤 며칠 시점까지 스튜디오로 돌아와 합쳐서 N시간 이상 다시 작업했나"의 비율(%)

지속 작업률 (퍼블리시 후 30일 내)

첫 퍼블리시 후 30일 내 동일 월드 v2+ 업데이트 또는 다른 월드 신규 퍼블리시한 비율. 퍼블리시 코호트별

이탈 후 복귀 크리에이터

마지막 스튜디오 로그인 30일+ 경과 후 재로그인한 크리에이터 (활성화 기준 재충족). 월별 복귀 수

Partner(외주) 계약자 정착률

계약 완료일로부터 30일 내 5시간+ 재활동 · 단기 핵심 보조지표
